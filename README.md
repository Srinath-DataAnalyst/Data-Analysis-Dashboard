# HR-Data-Analysis (Interactive Dashboard creation using Power BI)
## Project Objective
This project analyzes HR data using Power BI to uncover insights related to employee demographics, hiring, and attrition trends. The dashboard enables HR teams to make strategic decisions and improve workforce planning.

# Dataset used
- <a href="https://github.com/Srinath-DataAnalyst/Data-Analysis Dashboard/blob/main/Employee%20Sample%20Data.xlsx"> Dataset</a>

## Questions (KPIs)
•	Illustrate Headcount breakdown by Business Unit. 
•	Display Headcount distribution across various age groups. 
•	Analyze Headcount based on gender. 
•	Breakdown of Headcount by Job Title. 
•	Show Headcount distribution across different departments. 
• Analyze employee tenure distribution.
• Display employee headcount trend over time (monthly/quarterly/yearly).
•	How has employee hiring changed over the years, and is there a noticeable upward or downwar   trend in recruitment?

- Dashboard Interactive <a href="https://github.com/Srinath-DataAnalyst/Data-Analysis- Dashboard/blob/main/HR%20analysis.png">View Dashboard</a>

## Process
Verify the data for any gaps, errors, or inconsistencies, and clean or correct them.
Load the cleaned dataset into your dashboard tool
Use a measure/formula (e.g., NOW() in Power BI) to capture the last refresh date and time.
Insert a card or text box to display the timestamp on the dashboard.
Refresh the data to ensure the timestamp updates correctly.
